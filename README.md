# Slack-Bot
Learning how to make a slack bot

Current commands:
- /harmonic-bot-ping: Returns with a Pong and latency.
- /uwuify: Provide input and the bot will return an 'uwuified' version
- /stoic-quote: Returns a stoic quote, and its author
- /eight-ball: Returns an eight-ball fortune.

Ai Usage:
- None
